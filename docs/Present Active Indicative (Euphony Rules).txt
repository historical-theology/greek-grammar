Derivation & Euphony Rules (Smyth §463–464)

	1s	λυω < λυ+{ω} < λυ+{ο} 
		- final stem volwel ο lengthens to ω [can't site why]

	2s	λυεις < λυ+ει{ς} < λυ+ει{+} < λυ+ει < λυ+ε{-}ι < λυ+ε{σ}ι 
		- σ ending is added analogically: λυεσι (homeric) > λυεις (attic)

	3s 	λυει < λυ+ει < λυ+ε{-}ι < λυ+ε{σ}ι < λυ+ε{τ}ι
		- τ changes to σ, and σ drops out: λυε{τ}ι (doric) > λυε{σ}ι (homeric)
